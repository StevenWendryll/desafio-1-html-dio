Explicando algumas Tags

Abaixo teremos algumas tags mais comuns usadas em HTML e seus respectivos exemplos, no site da w3schools você pode saber um pouco mais a fundo sobre tags HTML e suas aplicações.

<h1>até <h6>

São tags usadas para definir títulos e subtítulos em uma página HTML, sendo <h1> o mais importante e <h6> o menos importante.

<p>

É a tag usada para definir parágrafos em uma página HTML.

<p>Este é um parágrafo de texto.<p>

<br>

A tag <br> é usada em HTML para inserir uma quebra de linha. Diferente das tags de parágrafo <p>, que criam um bloco de texto com espaço antes e depois, a tag <br> simplesmente move o texto que vem depois dela para a próxima linha sem adicionar espaço extra.

<mark>

É utilizada para destacar um trecho de texto, geralmente com um fundo amarelo para chamar a atenção do leitor.

Este é um texto com marcação para destacar uma parte específica.

<small>

Essa tag é usada para definir texto de tamanho menor, muitas vezes para informações secundárias.

Este é um texto normal. Este texto é menor.

<i>

Indica texto em itálico, embora seu uso atualmente seja mais voltado para ênfase semântica do que para itálico visual.

Este texto está em itálico.

<u>

Usado para sublinhar texto, embora não seja mais recomendado em termos de design web moderno, pois sublinhado geralmente indica um link.

Este texto está sublinhado.

<strong>

Usado para dar ênfase forte a um texto, geralmente exibido em negrito.

Este texto está em negrito.

<ol>

Cria uma lista ordenada (numerada), onde cada item é precedido por um número.

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

<ul>

Cria uma lista não ordenada (com marcadores), onde cada item é precedido por um marcador.

  • Primeiro item
  • Segundo item
  • Terceiro item

<li>

Define um item em uma lista (<ol> ou <ul>).

  • Item da lista

<a>

É usada para criar um link para outra página ou recurso, e pode conter o atributo href que especifica o endereço do link.
Exemplo de link

<hr>

Cria uma linha horizontal para separar conteúdo.

<sub>

Define texto como subscrito, que é exibido abaixo da linha de base.

Este é um exemplo de texto com H2O, onde o "2" está em subscrito.

<sup>

Define texto como sobrescrito, que é exibido acima da linha de base.

Este é um exemplo de texto com E=mc2, onde o "2" está em sobrescrito.

<blockquote>

É usado para definir um bloco de citação, geralmente com recuo em relação ao restante do texto.

"Este é um exemplo de citação longa."

<font>

Esta tag é usada para definir a cor, tamanho e fonte do texto.

Este é um texto estilizado com a tag.

<del>

É usada para indicar texto que foi deletado ou removido do documento.

Este produto custava $50,00 agora custa $30,00.

<abbr>

É usada para definir uma abreviação ou acrônimo, passe o mause sobre a palavra 'ONU' para visualizar.

A sede da ONU está localizada em Nova York.
